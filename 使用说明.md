# 华耀东方 · 成长印记 — 使用说明

## 网站简介

**华耀东方 · 成长印记** 是华耀东方女性全维成长平台的活动记录中心，统一汇总和展示开课、沙龙、赋能、内训等各类活动。

品牌核心理念：**东方的崛起，首先是东方女性的觉醒·崛起**

### 设计风格

- **主色调**：深紫色系（#14081A 深紫黑 + #6B2D7D 皇家紫），体现女性力量与高级感
- **点缀色**：香槟金（#C9A961），增添奢华质感
- **字体**：Noto Serif SC（衬线体，用于标题）+ Noto Sans SC（无衬线体，用于正文）
- **设计灵感**：YSL高级时装风格，深色背景与金色点缀的奢华美学

### 活动分类

- **开课** — 女性创业者演说力、形象影响力等课程
- **沙龙** — 行业交流、主题分享沙龙
- **赋能** — 合伙人赋能大会、市场赋能活动
- **内训** — 新员工培训、销售特训等内部训练

## 文件结构


## 如何使用

### 1. 查看网站

直接用浏览器打开 `activity-recorder.html` 文件即可。

### 2. 自动化更新（推荐方式）

您只需要按以下格式发送活动信息，我会自动完成全部更新：

```
活动类型：[开课 / 沙龙 / 赋能 / 内训]
活动日期：YYYY-MM-DD
活动地点：xxx

活动素材：（上传图片/视频）

活动内容简述：
（简要描述活动主题、参与人员、主要内容）
```

**我会自动完成：**

1. 分析素材内容，生成符合华耀东方品牌调性的专业文案
2. 提取活动亮点（参与人数、核心环节、成果数据等）
3. 生成合适的标签
4. 将图片保存到 `assets/images/` 目录
5. 更新 `assets/data.js` 数据文件
6. 您刷新网页即可看到新活动

### 3. 手动更新

编辑 `assets/data.js`，在 `activities` 数组中添加新对象：

```javascript
{
  id: "YYYYMMDD_XXX",
  title: "活动标题",
  category: "course",          // course/salon/empower/training
  date: "2026-07-10",
  location: "活动地点",
  description: "活动描述文案",
  images: ["assets/images/xxx.jpg"],
  videos: [],
  highlights: ["亮点1", "亮点2"],
  tags: ["标签1", "标签2"]
}
```

## 注意事项

1. **图片路径**：活动图片请存放在 `assets/images/` 目录下，使用相对路径引用
2. **数据备份**：建议定期备份 `assets/data.js` 文件
3. **品牌一致性**：文案风格应体现女性成长、觉醒、赋能的品牌调性
4. **网站部署**：将整个文件夹上传至服务器或托管平台（GitHub Pages、Vercel等）即可在线访问
